A **lambda** in C++ is a small, inline, anonymous function that you can define directly where you need it‚Äîwithout giving it a formal name.

They were introduced in **C++11** and are heavily used in modern C++ for callbacks, algorithms, and functional-style programming.

---

# üîç **What a Lambda Is ‚Äî The Simple Explanation**

A **lambda expression** is a quick way to write a function *inside* your code.

Example:

```cpp
auto add = [](int a, int b) {
    return a + b;
};
cout << add(2, 3);   // prints 5
```

Here:

* `[]` is the capture list
* `(int a, int b)` are the parameters
* `{ return a + b; }` is the function body

Lambdas behave just like small, inline functions.

---

# üìå **Why Lambdas Exist**

Before lambdas, you had to create functor structs just to pass functions:

```cpp
struct Add {
    int operator()(int a, int b) { return a + b; }
};
```

Now you can do it in one line.

---

# üß† **How Lambdas Work Internally**

Under the hood, a lambda generates a compiler-created class with:

* an `operator()` (call operator)
* member variables that store captured variables

This is why lambdas can capture local variables.

---

# ‚úã **Capture List (the `[]` part)**

The capture list defines which outside variables the lambda is allowed to use:

### 2Ô∏è‚É£ Capture by value (copy)


### 3Ô∏è‚É£ Capture by reference

```cpp

```

### 4Ô∏è‚É£ Capture all by value

```cpp

```

### 5Ô∏è‚É£ Capture all by reference

```cpp

```

---

# ‚ö° **`mutable` ‚Äî Why It Exists**

By default, captured-by-value variables are **read-only** inside the lambda.

Example:

```cpp
int x = 10;
auto f = [x]() { x++; };  // ‚ùå error: cannot modify captured copy
```

To modify the *copy*, use `mutable`:

```cpp
auto f = [x]() mutable { x++; };
```

Important: this still does NOT modify the original `x`.

---

# üåÄ **Lambda + `auto` + STL**

Lambdas work beautifully with STL algorithms:

```cpp
vector<int> v = {1, 2, 3, 4};

for_each(v.begin(), v.end(), [](int x){
    cout << x << " ";
});
```

Or filtering:

```cpp
auto it = find_if(v.begin(), v.end(),
                  [](int x){ return x % 2 == 0; });
```

---

# üìã **Full Lambda Syntax Breakdown**

```
[captures] (parameters) -> return_type {
    body
};
```

Example:

```cpp
auto f = [x](int y) -> int {
    return x + y;
};
```

Return type can be omitted if obvious.

---

# üß© **Real Uses of Lambdas**

* Callbacks (GUI, networking, events)
* Sorting with custom rules
* Thread functions
* Functional-style data processing
* Replacing functor classes

Example: sorting by length

```cpp
sort(v.begin(), v.end(),
     [](const string& a, const string& b){
         return a.size() < b.size();
     });
```
